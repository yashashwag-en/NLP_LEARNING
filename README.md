#NLP

This repository is where I’m learning and experimenting with Natural Language Processing (NLP) step by step.
The goal is not just to use libraries, but to actually understand how text data is processed, represented, and used to build intelligent models.

I started this while exploring machine learning and wanted a dedicated place to try small ideas, test concepts, and keep improving my understanding of NLP through practical code.

What you’ll find here

Text preprocessing (cleaning, tokenization, stopwords, etc.)

Feature extraction techniques like Bag of Words and TF-IDF

Basic machine learning models for text classification

Small experiments and practice notebooks while learning NLP concepts

This repo will keep growing as I learn more and try advanced topics like embeddings, deep learning models, and real-world NLP applications.

Tech used
 Google Colab

Why I made this

I believe the best way to learn something is by building and experimenting, not just reading theory.
This repo is basically my learning space where I document progress through code.
